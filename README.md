# CAT 🐱 — Coding Agentic Terminal

CAT is a production-grade, fully featured agentic terminal AI powered by **Minimax M2.7** via the **NVIDIA API**. It features an animated mascot, a rich TUI, and 20 powerful tools for autonomous software engineering.

## 🚀 Quick Start

### Installation

**Windows (PowerShell):**
```powershell
irm https://raw.githubusercontent.com/WyverncW/cat-cli/main/install.ps1 | iex
```

**macOS/Linux (Bash):**
```bash
curl -fsSL https://raw.githubusercontent.com/YOUR_USERNAME/cat-cli/main/install.sh | bash
```

### Setup

On first run, Cat will ask for your **NVIDIA API Key**.
You can get one here: [NVIDIA API Catalog](https://integrate.api.nvidia.com)

## 🛠️ Features

- **Animated Mascot:** A lifelike ASCII cat that reacts to the AI's state.
- **Agentic Loop:** Autonomous "Think → Act → Observe" cycle.
- **20 Tools:** Bash, File editing, Python, Web search, and more.
- **Project Context:** Proactively reads `CAT.md` for project-specific instructions.
- **Memory:** Remembers your preferences and project details.

## ⌨️ Slash Commands

- `/init` - Create project instructions (`CAT.md`)
- `/help` - Show full command list
- `/clear` - Start a fresh conversation
- `/memory` - Manage stored facts
- `/doctor` - Run system diagnostics
- `/exit` - Quit Cat
